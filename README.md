# My-Portfolio

웹디자이너 김주연의 개인 포트폴리오 웹사이트입니다.
디자인부터 퍼블리싱까지 직접 작업했으며, 프레임워크 없이 **HTML/CSS/JavaScript**로만 제작했습니다.

🔗 **[사이트 바로가기](https://your-domain.com)** ← 배포 후 실제 주소로 교체해주세요

---

## 📌 소개

Figma로 디자인한 시안을 html,css,javascrip로 직접 퍼블리싱한 포트폴리오입니다.
반응형 레이아웃, 인터랙션, 애니메이션까지 라이브러리 없이 순수 CSS/JS로 구현했습니다.

## 🛠 사용 기술

| 분류 | 기술 |
| --- | --- |
| 마크업 / 스타일 | HTML5, CSS3 (Flexbox, Grid, CSS 변수) |
| 스크립트 | Vanilla JavaScript (ES6+) |
| 폰트 | Pretendard, Oxanium, SB Aggro |
| 디자인 | Figma |

## ✨ 주요 기능

- **반응형 레이아웃** — 모바일부터 대형 모니터까지 대응
- **인터랙티브 히어로 섹션** — CSS 애니메이션으로 표정이 바뀌는 로고 인터랙션
- **무한 슬라이드 툴 아이콘** — 화면 폭에 맞춰 자동으로 개수를 계산해 끊김 없이 반복 (JS)
- **프로젝트 쇼케이스** — AERHER, LAFTEL, ILOOM, CGV 4개 프로젝트 소개
- **디자인 아카이브 페이지** — 매서너리 그리드 + 이미지 캐러셀 모달
- **맨 위로 가기 버튼** — 스크롤에 따라 자동 표시/숨김
- **메일 보내기** — Gmail / 네이버메일 / 다음메일 중 선택해서 바로 작성 화면으로 이동
- **전화번호 복사** — 클릭 시 클립보드 복사 (모바일에서는 전화 연결)
- **이력서 다운로드** — 헤더/푸터에서 PDF 즉시 다운로드

## 📁 폴더 구조

```
├── index.html              # 메인 페이지
├── design.html             # 디자인 아카이브 페이지
├── resume.pdf              # 이력서
├── css/
│   ├── style.css           # 메인 스타일
│   ├── design-archive.css  # 디자인 아카이브 스타일
│   └── sb-aggro.css        # SB Aggro 폰트 @font-face
├── js/
│   ├── script.js           # 메인 인터랙션
│   ├── design-data.js      # 디자인 아카이브 데이터
│   └── design-archive.js   # 디자인 아카이브 인터랙션
└── images/                 # 이미지 에셋
```
